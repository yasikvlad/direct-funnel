# Direct Funnel Plugin

Плагин-конструктор воронки продаж услуг. Помогает эксперту или специалисту создать полную цепочку: от сильного оффера до бот-сообщений, которые выдают лидмагнит и ведут на заявку.

---

## Как работает

Воронка состоит из 6 последовательных этапов:

1. **Докрутка оффера** — превращает сырое описание услуги в сильный, результато-ориентированный оффер
2. **Байт под вирусный контент** — пишет концовку для поста/рилса/карусели, которая органично ведёт к лидмагниту
3. **Сбор данных для лидмагнита** — задаёт 5 вопросов для написания материала (или принимает готовый)
4. **Написание лидмагнита** — статья (800–1500 слов) или мини-гайд (400–800 слов) с реальной ценностью
5. **Проверка CTA** — убеждается, что в лидмагните есть призыв на заявку/консультацию
6. **Тексты для бота** — цепочка из 3 сообщений: выдача → CTA → дожим

---

## Команды

| Команда | Что делает |
|---------|-----------|
| `/direct_funnel` | Запускает полный сценарий воронки с нуля |
| `/direct_funnel [твой оффер]` | Запускает воронку, начиная с готового оффера |

---

## Скилы (можно использовать отдельно)

| Скил | Когда использовать |
|------|-------------------|
| `direct-funnel` | Полный сценарий воронки — все 6 этапов |
| `offer-sharpener` | Только докрутка оффера |
| `lead-magnet-writer` | Только написание лидмагнита (статья или мини-гайд) |
| `bot-sequence-writer` | Только тексты для бот-цепочки |

---

## Примеры триггеров

**Полная воронка:**
- «создай воронку для моих услуг»
- «помоги с воронкой продаж»
- `/direct_funnel я помогаю дизайнерам выйти на 200к`

**Только оффер:**
- «докрути мой оффер»
- «сделай оффер сильнее»

**Только лидмагнит:**
- «напиши лидмагнит для коучей»
- «создай мини-гайд под мою тему»

**Только бот-тексты:**
- «напиши тексты для бота»
- «напиши цепочку сообщений для выдачи лидмагнита»

---

## Настройка

Плагин не требует дополнительной конфигурации.

Для сохранения результатов в Notion — подключи Notion-коннектор в настройках Cowork. Плагин предложит сохранить в конце каждой сессии.
