# Шаблоны Instagram DM-бота

## Принципы Instagram DM

- Короче, чем Telegram — люди в Instagram читают быстро
- Первое сообщение: должно вызвать желание получить лидмагнит и подписаться 
- Второе сообщение (подтверждена подписка): перевести с высокой конверсией на следующий шаг
- Кнопки — ключевой элемент. Текст кнопки = очевидное действие
- Пиши триггерно, цепляюще, но без лишних букв, краткими цепляющими фразами.

## Сообщение 1 — Первоначальный контакт

Отправляется: сразу после того, как человек написал кодовое слово

### Структура
1. Цепляющий триггерный заголовок (1 строка — не «привет», а что-то по теме)
2. Тизеры, что именно внутри лидмагнита. Не «узнаешь всё про агентов» — а конкретная механика, кейс, результат
3. Один шаг до получения: подписаться на аккаунт
4. Кнопка

### Шаблон А (энергичный)
```
Здесь всё про [тема лидмагнита коротко] 

Внутри:
→ [конкретный инсайт 1 из лидмагнита]
→ [конкретный инсайт 2 из лидмагнита]
→ [конкретный инсайт 3 из лидмагнита]

Чтобы получить — подпишись на меня и нажми кнопку ниже 👇

[Кнопка: Я подписан(а), проверить ✅]
```

### Шаблон Б (история/шок)
```
[Живая зацепка — результат, который удивит. Пример: «В шоке от того, что только что сделал мой агент»]

[2–3 строки — что конкретно произошло, с числами]

Всё это доступно тебе прямо сейчас — держи ссылку на материал.

Один шаг: подпишись на на меня и нажми кнопку ниже 👇

[Кнопка: Проверить подписку ✅]
```

### 🥇 Золотой стандарт (реальный пример А — лаконичный)
```
Здесь всё про нейросеть, которая помогает анализировать контент, собирать карусели и писать тексты без шаблонов

[Кнопка: открыть материал]
```

### 🥇 Золотой стандарт (реальный пример Б — шок + список результатов)
```
Честно, я в шоке (если мягко выражаться)

Смотрите что только что сделал мой агент по контенту.

Я закинула ему транскрипцию одного зума.
Ушла, вернулась и он сделал:
﻿﻿6 сценариев для Reels
﻿﻿2 карусели с готовыми текстами для карточек
﻿﻿3 коротких поста
﻿﻿2 длинных поста
﻿﻿2 серии Stories
﻿﻿8 цитат
﻿﻿1 статью

Итого: 16 единиц контента из одной транскрипции, которые он сделал полностью сам

[Кнопка: ИИ-агент тут]
```

### 🥇 Золотой стандарт (реальный пример В — провокационный вопрос + кнопка)
```
Как сделать 50k аудитории c 1 рилс?
Стратегия как набирать аудиторию в 2026 году

Жми на кнопку чтобы получить👇

[Кнопка: получить]
```

### 🥇 Золотой стандарт (реальный пример Г — личный ассистент, «только представьте»)
```
Пора создать своего ии-ассистента, который будет писать под ваш стиль речи (или докрутить своего моими фишками, если он работает криво)

Только представьте:
...он заменит вам 1000 попыток заставить ИИ перестать писать шаблонную херню.
...он поможет множить ваши мысли в разы быстрее. Таким образом, что вы сможете вести несколько соц.сетей, тратя меньше времени.
...он будет на связи 24/7, будет вашей правой рукой и ему не надо каждый раз объяснять заново.

Все инструкции в моем тг-канале, забирай их

[Кнопка: Создать ии ассистента]
```

## Сообщение 2 — Подписка подтверждена ✅

Отправляется: когда бот проверил и видит подписку

### Структура
1. Подтверждение подписки (1 строка)
2. **3 буллита с тизерами** — другие углы, чем в сообщении 1
3. **Личная строка от автора** — почему этот материал важен (1 предложение, без пафоса)
4. Ссылка на Telegram-бот
5. Короткий призыв открыть

### Шаблон А
```
Забирай [название] по кнопке ниже.

Внутри:
→ [тизер 1 — другой угол, чем в сообщении 1]
→ [тизер 2]
→ [тизер 3]

[Личная строка: почему этот материал важен для автора / что внутри особенного, что ценного это даст пользователю]

Открывай в Telegram 👇

[Кнопка: Получить в Telegram]
```

### Шаблон Б (лаконичный)
```
[1 провокационный вопрос, который есть в голове у пользователя в теме автора, например "как набирать аудиторию в 2026 году", или "как создать своего ии ассистента, который освободит 3 часа в неделю"]

Вижу подписку, забирай [инструкцию / видео и тд]
👇

[Кнопка: Открыть материал]
```

## Сообщение 3 — Не подписан ❌ (первая проверка)

Отправляется: когда бот проверил и подписки нет

### Шаблон А
```
Пока не вижу тебя в подписчиках ... 

Подпишись и нажми кнопку снова, проверю ещё раз 👇

[Кнопка: Я подписался(ась), проверить]
```

### Шаблон Б (мягкий)
```
Хм, подписка пока не отображается.

Зайди на мой аккаунт → подпишись → вернись и нажми кнопку ниже.

[Кнопка: Готово, проверь]
```

## Сообщение 4 — Не подписан ❌ (повторная проверка)

Отправляется: когда человек нажал повторно, но подписки всё ещё нет

### Шаблон А
```
Всё ещё не вижу подписку.

Иногда Инстаграм показывает с задержкой — попробуй через минуту 🙏

[Кнопка: Проверить ещё раз]
```

### Шаблон Б (с помощью)
```
Что-то не срабатывает 🤔

Иногда нужна пара минут. Попробуй:
1. Зайди на мою страницу 
2. Убедись, что подписка активна (кнопка «Подписан»)
3. Вернись и нажми ниже

[Кнопка: Готово, проверь]
```

## Советы по настройке DM-бота

**Кнопки:** Используй не более 1–2 кнопок на сообщение. Один выбор = меньше трения.

**Триггеры:** Настрой реакцию на кодовое слово в любом регистре (АГЕНТ / агент / Агент).

**Задержки:** Не нужны в первых сообщениях. Бот должен отвечать мгновенно.

**Дополнительные триггеры для Сообщения 2:** Некоторые платформы (Manychat, BotHelp) позволяют автоматически нажимать «Проверить подписку» через N секунд после Сообщения 1 — это улучшает конверсию.
