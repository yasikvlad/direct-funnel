# Гайд по написанию байта (концовки, ведущей к лидмагниту)

## Что такое байт в контексте воронки

Байт — это 1–3 предложения в конце вирусного контента, которые:
1. Органично вытекают из темы поста/рилса/карусели
2. Называют лидмагнит как естественный следующий шаг
3. Дают чёткое действие: что написать, куда нажать, что сказать

Байт не должен выглядеть как реклама. Он — логическое продолжение ценности.

---

## 6 паттернов байта

### Паттерн 1: «Хочешь глубже»
Когда контент даёт инсайт, но не весь путь целиком.
```
Если хочешь разобрать это глубже — я написал(а) [название лидмагнита].
Там [что внутри в 1 предложении]. Напиши [кодовое слово] — пришлю.
```

### Паттерн 2: «Следующий шаг»
Когда контент описывает проблему, лидмагнит — решение.
```
Если ты сейчас на этом этапе — у меня есть [название].
[Что получит читатель]. Пиши [слово] в директ / ответным сообщением.
```

### Паттерн 3: «Это только часть»
Когда контент — один из шагов большой темы.
```
Это лишь один из [N] шагов. Остальные — в [название лидмагнита].
Забирай: напиши [слово].
```

### Паттерн 4: «Я собрал(а) это для тебя»
Личный, тёплый тон. Подходит для экспертов с близкой аудиторией.
```
Я собрал(а) [что именно] в отдельный [формат].
Без воды, только суть. Пиши [слово] — отправлю.
```

### Паттерн 5: «Проверь себя»
Когда контент про диагностику или ошибки.
```
Хочешь проверить, как это работает у тебя? В [название] — [что внутри].
Пиши [слово].
```

### Паттерн 6: «Сохрани и используй»
Когда контент практичный — инструкция или чеклист.
```
Сохрани этот пост. А ещё — у меня есть [название]:
[что там]. Получи по слову [слово].
```

---

## Кодовые слова

Кодовое слово — то, что пользователь пишет боту, чтобы получить лидмагнит.
Должно быть коротким, связанным с темой, легко запоминаемым.

Примеры: «ГАЙД», «СТАРТ», «ХОЧУ», «ДА», «ПЛАН», «ЧЕКЛИСТ», тематическое слово.

---

## Частые ошибки в байтах

- Слишком длинно — байт больше 4 предложений теряет силу
- Нет конкретного действия — "посмотри мой профиль" не работает
- Слабое название лидмагнита — не цепляет → не нажимают
- Байт не связан с темой контента — выглядит как случайная реклама
