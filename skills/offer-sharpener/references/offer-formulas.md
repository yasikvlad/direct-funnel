# Формулы сильного оффера

## Базовые формулы

### Формула 1: Результат + Аудитория
```
[Глагол действия] [кому] → [конкретный результат]
```
Пример: «Помогаю фрилансерам выйти на 150к без холодных продаж»

### Формула 2: Результат + Аудитория + Без боли
```
[Результат] для [кого] без [главная боль/страх]
```
Пример: «Стабильный поток клиентов для коучей без соцсетей и рилсов»

### Формула 3: Трансформация
```
[Откуда] → [Куда] для [кого]
```
Пример: «Из "работаю за копейки" в "выбираю клиентов сам" — для веб-дизайнеров»

### Формула 4: Процесс + Результат + Срок
```
[Что делаю] → [результат] за [время]
```
Пример: «Упаковываю эксперта под ключ — сайт, воронка, контент — за 3 недели»

### Формула 5: Проблема → Решение
```
Если [боль/ситуация] — помогаю [решение/результат]
```
Пример: «Если клиенты "уходят думать" — выстраиваю систему, где они покупают сами»

---

## Усилители оффера

Добавляй к любой формуле:
- **Цифры**: «за 21 день», «100+ клиентов», «в 3 раза больше»
- **Без/Без того**: «без холодных звонков», «без сложных воронок», «без выгорания»
- **Даже если**: «даже если нет аудитории», «даже если только начинаешь»
- **Гарантия**: «или верну деньги», «или бесплатно переделаю»

## Частые ошибки в офферах

| Плохо | Почему | Лучше |
|-------|--------|-------|
| «Я коуч по личностному росту» | Нет результата, нет аудитории | «Помогаю предпринимателям выйти из операционки и зарабатывать больше работая меньше» |
| «Делаю сайты» | Слишком широко, нет ценности | «Делаю сайты для экспертов, которые продают сами — без SEO и рекламы» |
| «Комплексный маркетинг» | Непонятно, для кого | «Строю маркетинг для B2B-компаний, которым нужны клиенты, а не просто охваты» |
| «Помогаю бизнесу расти» | Абстрактно | «Удваиваю выручку малого бизнеса за 6 месяцев через систему повторных продаж» |

## Тест сильного оффера

Сильный оффер должен пройти 3 вопроса:
1. Понятно ли, ДЛЯ КОГО это — после первого прочтения?
2. Понятен ли РЕЗУЛЬТАТ — конкретно, без интерпретации?
3. Есть ли причина выбрать именно этого человека (а не другого)?

Если хотя бы один ответ «нет» — оффер нужно переписать.
